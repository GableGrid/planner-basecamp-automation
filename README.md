# Planner to Basecamp Automation 🔄

## Overview
Automatically syncs new Microsoft Planner tasks to Basecamp 3 using Power Automate.

## How It Works
1. New task created in Microsoft Planner
2. Power Automate trigger fires
3. Task automatically created in Basecamp 3

## Tools Used
- Microsoft Planner
- Power Automate
- Basecamp 3

## Fields Mapped
- Task Title
- Due Date

## Testing
Tested with 8+ tasks — all synced successfully ✅

## Error Handling
Email notification sent to mefta@gablegrid.com if sync fails
